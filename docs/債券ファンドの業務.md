# 債券ファンド・デスクの業務と収益源

このチュートリアルで学ぶ技術が、実際の債券運用・トレーディングの現場でどう使われ、
どこでお金が生まれるのかを整理する。各 notebook 冒頭の「実務での位置づけ（ファンドは
どう稼ぐか）」はこのドキュメントを土台にしている。

## 誰が債券を扱うか

債券の担い手は大きく売り手（セルサイド）と買い手（バイサイド）に分かれる。

| 区分 | 主体 | 役割 | 主な収益源 |
|---|---|---|---|
| セルサイド | 投資銀行のトレーディングデスク | マーケットメイク（値付け） | ビッド・オファースプレッド、フロー |
| バイサイド | ヘッジファンド | 自己資金で方向・相対価値に賭ける | 価格変動・裁定・キャリー |
| バイサイド | 運用会社・年金・保険 | 顧客資産の運用・ALM | 運用報酬、超過リターン（アルファ） |

セルサイドのトレーダーは担当商品の値付け役（マーケットメイカー）で、在庫を持ちながら
買値と売値を提示し、その差（スプレッド）で稼ぐ。顧客との窓口はセールスが持ち、顧客の
売買ニーズ（RFQ）をトレーダーに取り次ぐ。

## デスクの分かれ方：レート vs クレジット

- **レートデスク**は「マクロ」を見る。経済成長・インフレ・金融政策・為替が対象で、
  国債・金利スワップ・金利先物・スワップション等の金利商品をつくる。
- **クレジットデスク**は「ミクロ」を見る。個社の信用力が対象で、社債・CDS 等を扱う。

このチュートリアルは主にレート側（S1〜S6, S9, S10）を扱い、クレジット（S7）と
証券化（S8）も押さえる。

## お金の稼ぎ方（収益源）

### 1. マーケットメイク（スプレッド収益）

継続的に両面クォートを出し、顧客の買いに売り・売りに買って、ビッド・オファー
スプレッドを収益にする。約定のたびに在庫を抱えるため、在庫リスクとスプレッド収益の
綱引きを値付けに落とし込む。→ S11-2 RFQプライサー、S9-4 先物ベーシス。

### 2. 相対価値（レラティブバリュー, RV）

証券間の価格のゆがみが将来「収束」または「発散」する方向に賭ける。割安なキャッシュ
フローを買い割高を売ると、ヘッジ後のキャリー（利ざや）が持続的なアルファになる。
代表的な RV は次のとおり。

- **イールドカーブ取引**（スティープナー・フラットナー・バタフライ）→ S9-2
- **銘柄の rich/cheap**（フィット済カーブからの乖離の平均回帰）→ S9-1
- **先物ベーシス・CTD**（先物と現物の裁定）→ S9-4
- **統計裁定・平均回帰**（正常な関係からの統計的乖離）→ S9-1, S9-5

### 3. キャリー戦略

保有しているだけで得る利息と、それを賄うファンディング（レポ）費用の差で稼ぐ。
国債中心の RV ファンドは NAV の 40〜60% を未拘束現金として持ち、その現金の金利
（短期国債等）も収益にしつつ、レポとデリバティブでレバレッジをかける。→ S9-3。

### 4. マクロ

金利・為替・インフレの大きな方向観に賭ける。流動性の高い OTC 商品を使い、NAV の
50〜70% を短期国債等で運用しながらポジションを取る。→ S3（リスク・シナリオ）、
S5〜S6（モデル・デリバティブ）。

### 5. 運用会社・ALM（超過リターンとリスク管理）

年金・保険は負債に対して資産を組む ALM が中心で、デュレーション・キーレートで
免疫化する。パッシブ運用はインデックスを少数銘柄で複製し、アクティブ運用は
デュレーション・カーブ・クレジットの見立てでアルファを狙う。→ S10。

## レバレッジとファンディング

狙ったレバレッジは、プライムブローカー経由の担保付きレポ（現先取引）で作る。証券の
種類ごとにヘアカット（担保掛け目）がかかる。レポ金利がキャリーの符号を左右する。→ S9-3。

## クオンツ・リスクの役割

- **クオンツ（フロント）**: カーブ構築・モデルのキャリブレーション・デリバティブ評価。
  → S2, S5, S6。
- **リスク管理クオンツ**: DV01・KRD・VaR/ES・シナリオでポジションのリスクを測る。
  → S3。
- **モデルバリデーション**: フロントのモデルを独立に検証し承認する（FRB SR 11-7 等）。
  → S5-5。
- **XVA デスク**: カウンターパーティリスク（CVA）を価格化しヘッジする。→ S7-5。
- **電子取引・マーケットメイク**: RFQ の自動値付けと在庫管理。→ S11-2。
