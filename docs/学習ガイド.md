# 学習ガイド

このチュートリアルをどの順で、どれくらいの時間で進めるかの道しるべ。各 notebook は
「学習目標 → 実務での位置づけ（ファンドはどう稼ぐか）→ 理論 → スクラッチ実装 →
QuantLib 検証 → 実データ適用 → 演習 → 用語集」の順で読む。

## はじめに（最初の30分）

1. `README.md` を読み、環境を作る（`pip install -e ".[dev]"`、`export PYTHONPATH=$PWD`）。
2. `pytest -q` が緑になることを確認する（61 テスト）。
3. [`docs/債券ファンドの業務.md`](債券ファンドの業務.md) に目を通し、「誰が何で稼ぐか」の
   地図を頭に入れる。各 notebook の意義がここに接続する。
4. S0（環境・データ基盤）から始める。

## つまずいたら

- **数学（S4）で止まる**: S4-1 冒頭の前提チェック（正規分布・大数の法則・テイラー展開）に
  戻る。S4-2 の伊藤の公式は導出より「使い方」を先に。数値検証セクションだけ通せば S5 へ
  進める。
- **QuantLib 突合が合わない**: 引数順に注意（例 CIR は `ql.CoxIngersollRoss(r0, theta, k, sigma)`、
  Vasicek は `ql.Vasicek(r0, a, b, sigma)`）。ACT/ACT は債券では ISMA/Bond 変種を使う。
- **カーブが歪む**: `bootstrap_par` は等間隔テナー前提。非等間隔なら年次グリッドへ補間して
  から渡す（警告が出る）。
- **図が出ない/数値が違う**: 乱数は seed 固定。`PYTHONPATH=$PWD` と依存バージョン
  （`requirements-lock.txt`）を確認。
