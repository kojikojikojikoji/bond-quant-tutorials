# 用語集 — 債券の基礎（S1）

<a id="discount-factor"></a>
### 割引係数（discount factor）
- 定義: 将来の時点 $t$ に受け取る1単位キャッシュフローの現在価値。レートと年数と複利規約から一意に定まる。
- 数式: $DF(t) = (1 + r/m)^{-mt}$（$m$回複利）、$DF(t) = e^{-rt}$（連続複利）
- 関連: 複利規約, スポットレート, 現在価値
- 初出: S1-1

<a id="continuous-compounding"></a>
### 連続複利（continuous compounding）
- 定義: $m$回複利で複利回数 $m$ を無限大にした極限。割引を積分で書けるため確率金利モデルで標準的に使う。
- 数式: $DF(t) = e^{-rt}$、離散との変換は $r_c = m\ln(1 + r_m/m)$、$r_m = m(e^{r_c/m}-1)$
- 関連: 割引係数, 複利規約
- 初出: S1-1

<a id="spot-rate"></a>
### スポットレート（spot rate）
- 定義: 時点 $t$ に一括満期を迎えるゼロクーポンから逆算した金利。ゼロレートと同義。カーブ構築の基本量。
- 数式: $r = -\ln DF(t)/t$（連続複利）、$r = m(DF(t)^{-1/(mt)}-1)$（$m$回複利）
- 関連: 割引係数, パー利回り
- 初出: S1-1

<a id="present-value"></a>
### 現在価値（present value）
- 定義: 将来キャッシュフローを割引係数で現時点に引き直した価値。
- 数式: $PV = C \cdot DF(t)$
- 関連: 割引係数
- 初出: S1-1

<a id="day-count-convention"></a>
### 日数計算規約（day count convention）
- 定義: 2つの日付の間の年数 $\tau$ を求める際の、分子（日数の数え方）と分母（1年の日数）の取り決め。クーポン・経過利子・割引年数の基礎。
- 数式: $\tau(d_1,d_2) = \dfrac{\text{日数}(d_1,d_2)}{\text{基準日数}}$
- 関連: ACT/360, ACT/365F, 30/360, ACT/ACT
- 初出: S1-2

<a id="actual-360"></a>
### ACT/360（actual/360）
- 定義: 実日数を360で割る規約。分母が実年長より小さく年数が大きめに出る。USD/EUR のマネーマーケットと変動金利計算の標準。
- 数式: $\tau = \dfrac{d_2 - d_1}{360}$
- 関連: 日数計算規約
- 初出: S1-2

<a id="actual-365-fixed"></a>
### ACT/365F（actual/365 fixed）
- 定義: 実日数を365（固定）で割る規約。うるう年でも分母は366にしない。日本国債（JGB）、GBP マネーマーケット等で使用。
- 数式: $\tau = \dfrac{d_2 - d_1}{365}$
- 関連: 日数計算規約
- 初出: S1-2

<a id="30-360-us-bond-basis"></a>
### 30/360（30/360, US Bond Basis）
- 定義: 1か月30日・1年360日換算の規約。日が31なら30へ丸める月末調整を行い、各期のクーポンが同額になる。米国社債の慣行。
- 数式: $\tau = \dfrac{360(y_2-y_1)+30(m_2-m_1)+(d_2'-d_1')}{360}$
- 関連: 日数計算規約
- 初出: S1-2

<a id="actual-actual-isda"></a>
### ACT/ACT（actual/actual, ISDA）
- 定義: 実日数を各暦年の実日数で按分する規約。年をまたぐと年ごとに365/366で割って足す。米国債の利回り換算で使用。経過利子の ISMA/ICMA 版（実日数の期間比）は別で、債券の経過利子はそちらが標準（S1-4）。
- 数式: $\tau = \dfrac{n_{y_1}}{D_{y_1}} + (y_2-y_1-1) + \dfrac{n_{y_2}}{D_{y_2}}$（$D_y$は年の実日数）
- 関連: 日数計算規約, 経過利子
- 初出: S1-2

<a id="modified-following"></a>
### 修正翌営業日（modified following）
- 定義: 休日の日付を翌営業日へずらす営業日調整。ただし翌営業日が翌月にはみ出す場合は前営業日へ戻し、利払日を同一月内に留める。実務で最も一般的。
- 関連: 利払スケジュール, 休日カレンダー
- 初出: S1-2

<a id="coupon-schedule"></a>
### 利払スケジュール（coupon schedule）
- 定義: 満期から利払間隔ぶん遡って刻み（backward生成）、各日を営業日調整した利払日の列。カーブ構築・債券評価の入力になる。
- 関連: 修正翌営業日, 休日カレンダー
- 初出: S1-2

<a id="holiday-calendar"></a>
### 休日カレンダー（holiday calendar）
- 定義: 市場（国）ごとの非営業日の集合。土日に加え国民の祝日・年末年始等を含む。営業日調整とスケジュール生成の前提になる。
- 関連: 修正翌営業日
- 初出: S1-2

<a id="yield-to-maturity-ytm"></a>
### 最終利回り（yield to maturity, YTM）
- 定義: 債券の全キャッシュフローを単一利回りで割り引いた現在価値を市場価格に一致させる割引率。将来CF列に対する内部収益率（IRR）で、満期保有・同一利回りでのクーポン再投資・無デフォルトを前提にしたときの年率リターンに一致する。
- 数式: $P_{\text{dirty}} = \sum_{j=0}^{n-1} c_j (1+y/f)^{-(w+j)}$ を満たす $y$
- 関連: 単利利回り, 割引係数, パー債, 凸性
- 初出: S1-3

<a id="par-bond"></a>
### パー債（par bond）
- 定義: クーポン率と YTM が等しく、清算価格が額面ちょうど（100）になる債券。$c>y$ ならプレミアム債（>100）、$c<y$ ならディスカウント債（<100）。
- 関連: 最終利回り
- 初出: S1-3

<a id="simple-yield"></a>
### 単利利回り（simple yield）
- 定義: 年あたりクーポンと満期償還損益を単純に年割りし、価格で割った利回り。複利効果と再投資を考慮しない日本の店頭表示慣行。複利 YTM とは一般に一致せず、ディスカウント債では高め、プレミアム債では低めに出る。
- 数式: $y_{\text{simple}} = \dfrac{C_{\%} + (100-P)/T}{P}$
- 関連: 最終利回り
- 初出: S1-3

<a id="newton-s-method"></a>
### Newton法（Newton's method）
- 定義: 関数値と一階微分から接線の零点へ更新を繰り返す求根法。価格-利回りはほぼ単調・凸なので数反復で収束するが、初期値が悪いと $(1+y/f)\le0$ 側へ飛んで発散しうる。
- 数式: $y_{k+1} = y_k - \dfrac{P(y_k)-P^{*}}{P'(y_k)}$
- 関連: 凸性, 最終利回り
- 初出: S1-3

<a id="convexity"></a>
### 凸性（convexity）
- 定義: 価格-利回り曲線が下に凸である性質（$\partial^2 P/\partial y^2>0$）。デュレーションによる一次近似（接線）は常に真の価格より下側を通り、そのずれの主要項が凸性。利回り低下の値上がりが上昇の値下がりを上回る非対称を生む。
- 数式: $\dfrac{\partial^2 P}{\partial y^2} = \sum_j \dfrac{(w+j)(w+j+1)}{f^2}\, c_j (1+y/f)^{-(w+j+2)}$
- 関連: 最終利回り
- 初出: S1-3

<a id="accrued-interest"></a>
### 経過利子（accrued interest）
- 定義: 直前利払日から受渡日までの、まだ支払われていない利息。売り手に帰属し、買い手がクリーン価格に上乗せして支払う。債券では ACT/ACT-ISMA（Bond）変種、すなわち実日数の期間比で日割りするのが標準。
- 数式: $AI_{\text{ISMA}} = \dfrac{C}{f}\cdot\dfrac{d(t_{\text{prev}}, t_s)}{d(t_{\text{prev}}, t_{\text{next}})}$
- 関連: クリーン価格, ダーティ価格, 受渡日, ACT/ACT
- 初出: S1-4

<a id="clean-price"></a>
### クリーン価格（clean price）
- 定義: 経過利子を除いた気配価格。受渡日による鋸歯状の変動を取り除くため、銘柄間・時点間の比較に用いる。市場で表示されるのは通常この価格。
- 数式: $P_{\text{clean}} = P_{\text{dirty}} - AI$
- 関連: ダーティ価格, 経過利子
- 初出: S1-4

<a id="dirty-price"></a>
### ダーティ価格（dirty price）
- 定義: クリーン価格に経過利子を足した、買い手が実際に支払う受渡代金。評価・約定・会計はこの価格で行う。
- 数式: $P_{\text{dirty}} = P_{\text{clean}} + AI = \sum_{j \ge 0} c_j (1 + y/f)^{-(w+j)}$
- 関連: クリーン価格, 経過利子
- 初出: S1-4

<a id="t-1-settlement"></a>
### T+1決済（T+1 settlement）
- 定義: 約定日の翌営業日に資金と証券を受け渡す決済慣行。米国債（UST）・英国債（Gilt）が採用。JGB・Bund の T+2 と対比される。
- 関連: 受渡日
- 初出: S1-4

<a id="settlement-date"></a>
### 受渡日（settlement date）
- 定義: 資金と証券が実際に交換される日。経過利子は約定日ではなく受渡日を基準に計算する。決済慣行（T+1 / T+2）で約定日からの日数が決まる。
- 関連: T+1決済, 経過利子
- 初出: S1-4
