# 用語集 — イールドカーブ構築（S2）

定義の正はこのファイル。各 notebook 末尾の用語表はここへのリンクとして扱う。
項目は S2 の各 notebook（S2-1〜S2-5）の初出順にまとまっている。

### ブートストラップ（bootstrap）
- 定義: パー利回りやスワップレートなど市場がクオートする商品価格から、割引係数の列を
  短期から順に剥ぎ取る前進代入の手続き。各テナーのパー債が価格1になる条件を並べると
  下三角の連立方程式になり、上から一意に解ける。
- 数式: $DF(t_i) = \dfrac{1 - (c_i/f)\sum_{k<i} DF(t_k)}{1 + c_i/f}$
- 関連: 剥ぎ取り, パーレート, ゼロカーブ
- 初出: S2-1

### ゼロカーブ（zero curve）
- 定義: 各年限 $t$ の単一支払いに対するゼロレート $z(t)$ を並べたカーブ。割引係数と
  $DF(t)=e^{-z(t)t}$ で一対一に対応し、ブートストラップの出力にあたる。
- 数式: $z(t) = -\ln DF(t) / t$
- 関連: ゼロレート, 割引係数, フォワードレート
- 初出: S2-1

### フォワードレート（forward rate）
- 定義: いま約定し、将来区間 $[t_1, t_2]$ に適用される金利。無裁定から割引係数の比で
  定まる。区間を縮めた極限が瞬間フォワードで、ゼロレートはその累積平均。
- 数式: $f(t_1, t_2) = \dfrac{\ln DF(t_1) - \ln DF(t_2)}{t_2 - t_1}$
- 関連: ゼロカーブ, 割引係数
- 初出: S2-1

### パーレート（par rate）
- 定義: 年限 $t_n$ の標準クーポン債が額面（パー）で取引されるときのクーポン率。
  カーブ構築の入力になり、ゼロレートとは別物。右上がりカーブでは $c_n < z(t_n)$。
- 数式: $c_n = f\,\dfrac{1 - DF(t_n)}{\sum_{k=1}^{n} DF(t_k)}$
- 関連: ブートストラップ, ゼロカーブ, パー利回り
- 初出: S2-1

### 剥ぎ取り（strip）
- 定義: ブートストラップにおいて、短期テナーから順に割引係数を確定させ、確定分を
  running sum に引き継いで次のテナーへ進む逐次操作。短期の入力誤差は running sum を
  通じて長期ゾーンへ繰り越される。
- 関連: ブートストラップ, 割引係数
- 初出: S2-1

### Nelson-Siegel（Nelson-Siegel）
- 定義: ゼロレート曲線を、レベル・スロープ・曲率の3因子と1つの時定数 $\lambda$
  で表す4パラメータの関数形。少数の解釈可能なパラメータで曲線全体を要約する。
- 数式: $z(\tau)=\beta_0+\beta_1\frac{1-e^{-\tau/\lambda}}{\tau/\lambda}
  +\beta_2\left(\frac{1-e^{-\tau/\lambda}}{\tau/\lambda}-e^{-\tau/\lambda}\right)$
- 関連: Svensson, 因子負荷, パー利回り
- 初出: S2-3

### Svensson（Svensson）
- 定義: Nelson-Siegel に2つ目の曲率項（$\beta_3$ と時定数 $\lambda_2$）を足した
  6パラメータ拡張。山を2つ許し、より複雑なカーブ形状を表せる。NSS とも書く。
  ECB や FRB（GSW データセット）が公表カーブの推定に採用している。
- 数式: NS に $+\beta_3\left(\frac{1-e^{-\tau/\lambda_2}}{\tau/\lambda_2}
  -e^{-\tau/\lambda_2}\right)$ を加える
- 関連: Nelson-Siegel, 多重共線性
- 初出: S2-3

### 因子負荷（factor loading）
- 定義: 各因子を1単位動かしたときに、カーブが年限ごとにどれだけ動くかを表す
  感応度。ゼロレートの各 $\beta$ に関する偏微分に一致する。レベル負荷は定数1、
  スロープ負荷は短期で1・長期で0の単調減少、曲率負荷は両端0・中期に山。
- 数式: $\partial z/\partial\beta_0=1,\ \partial z/\partial\beta_1=L_1,\
  \partial z/\partial\beta_2=L_2$
- 関連: Nelson-Siegel, Svensson
- 初出: S2-3

### 非線形最小二乗（nonlinear least squares）
- 定義: パラメータが非線形に入るモデルで、残差二乗和を最小化する当てはめ。
  NSS では $\lambda$ が非線形に入るため、6変数同時最適化は局所解・初期値依存に
  陥りやすい。$\lambda$ を格子に固定して線形部分を OLS で解く2段階推定
  （概形最小二乗）で回避する。
- 数式: $\min_{\beta,\lambda}\sum_i (z(\tau_i)-y_i)^2$
- 関連: 多重共線性, Svensson
- 初出: S2-3

### 多重共線性（multicollinearity）
- 定義: 計画行列の列がほぼ平行になり、正規方程式 $X^\top X$ が特異に近づいて
  係数推定が不安定になる状態。NSS では $\lambda_1\approx\lambda_2$ のとき2つの
  曲率負荷がほぼ同形になり、$\beta_2,\beta_3$ が逆符号の大きな値へ発散する
  （曲線の当てはまりは良いのに係数が無意味になる）。
- 関連: 非線形最小二乗, 因子負荷
- 初出: S2-3

### ポイントインタイム（point-in-time）
- 定義: 各時点の計算に、その時点までに入手可能だった情報のみを使う原則。
  未来の値で欠損を埋める・将来の統計で当日を正規化する等の先読み
  （look-ahead）を排し、バックテストが実運用で再現可能であることを担保する。
  日次カーブ運用では前日比較・外れ値統計・フォールバックを「過去と当日」
  だけで組む。
- 関連: リグレッションテスト, 外れ値検出
- 初出: S2-5

### 外れ値検出（outlier detection）
- 定義: 入力に紛れる疑わしい点を機械的に警告する仕組み。日次カーブ運用では
  前日比の $z$ スコアが閾値 $k$（既定 $k=2$）を超えるテナーを警告する。
  市場が実際に大きく動いた日も引っかかるが、黙って通すより安全側に倒れる。
  閾値は感度と誤検知のトレードオフで決める運用パラメータ。
- 数式: $\left|(\Delta_i - \mu_i)/\sigma_i\right| > k,\quad
  \Delta_i = z_i^{(t)} - z_i^{(t-1)}$
- 関連: ポイントインタイム, フォールバック
- 初出: S2-5

### リグレッションテスト（regression test）
- 定義: 常に成り立つべき不変条件を毎回検証し、変更に伴う退行（regression）を
  検知する仕組み。カーブ構築では「パー再現」——構築後カーブでパー債を再評価
  すると価格が $1$ に戻る——が中核の不変条件になり、ブートストラップ実装や
  補間が壊れれば即座に破れる。
- 数式: $\dfrac{c_i}{f}\sum_{k \le i} DF(t_k) + DF(t_i) = 1$
- 関連: ユニットテスト, パーレート, ブートストラップ
- 初出: S2-5

### フォールバック（fallback）
- 定義: 入力検証やカーブ品質検査を通らない日に、例外で全体を止めず、直近で
  品質を通った最後のカーブ（last known good）を持ち越す縮退設計。運用系は
  「今日のカーブ」を必ず1本返す責務があるため、単純で信頼できる規則を1つに
  絞る。持ち越したことは必ずログに残し、後から人が追える状態にする。
- 関連: 外れ値検出, ポイントインタイム
- 初出: S2-5

### OIS（overnight index swap）
- 定義: 固定金利と、翌日物RFRを観測期間で複利した後決めレートを交換するスワップ。
  担保付取引の割引に使う担保金利の代理として、割引カーブ（OISカーブ）の構築入力に
  なる。OISは割引と推計が同一のRFRカーブなので、フロートレッグPVが $1-DF(T_n)$ に
  畳めて単一カーブとして剥ぎ取れる。
- 数式: フロートレッグPV $=1-DF_{ois}(T_n)$、パー条件 $K\sum_i DF_{ois}(T_i)=1-DF_{ois}(T_n)$
- 関連: RFR, デュアルブートストラップ, CSA
- 初出: S2-4

### SOFR/TONA（secured overnight financing rate / Tokyo overnight average rate）
- 定義: 米ドル・円の後決め翌日物RFR。SOFRは国債レポ（担保付）、TONAは無担保コール
  翌日物を対象とし、いずれもLIBORの後継。翌日物のためタームフォワードを直接は持たず、
  観測期間の複利で後決めタームレートを作る。
- 関連: RFR, OIS
- 初出: S2-4

### CSA（credit support annex, 信用補完契約）
- 定義: デリバティブ取引の担保のやり取り（値洗い・受渡し・担保金利）を定めるISDAの
  付属契約。差し入れた担保に付く担保金利が翌日物RFRであれば、担保付取引の割引は
  そのRFRを積んだOISカーブで行うのが整合的になる。OIS割引の論拠を与える。
- 関連: OIS, RFR
- 初出: S2-4

### RFR（risk-free rate, リスクフリーレート）
- 定義: パネル呈示に頼らず実取引に基づいて算出される翌日物金利。SOFR・TONA・SONIA
  など。LIBOR改革の後継金利で、割引（担保金利）と推計（フォワード）の双方の土台になる。
- 関連: SOFR/TONA, OIS, CSA
- 初出: S2-4

### デュアルブートストラップ（dual bootstrap）
- 定義: 割引カーブ（OIS）と推計カーブ（各インデックス）を2段で構築する手続き。
  第1段でOISのパーから $DF_{ois}$ を剥ぎ取り、第2段でそれを割引に固定したまま
  スワップのパーから $DF_{proj}$ を剥ぎ取る。第2段が割引と推計の2本を同時に使う
  ためデュアルと呼ぶ。末端ノードは閉形式で解ける。
- 数式: $DF_{proj}(n)=\dfrac{DF_{proj}(n-1)}{1+\text{RHS}/DF_{ois}(n)}$
- 関連: OIS, ブートストラップ, フォワードレート
- 初出: S2-4

### 凸性調整（convexity adjustment）
- 定義: 短期金利先物から読める先物レートと、カーブが必要とするフォワード（FRA）
  レートの差。先物は日々値洗いされ、金利と再運用コストの正相関が先物保有者に有利に
  働くため、先物レートはフォワードより高く付く。SOFR先物を短期ゾーンのカーブ入力に
  使う際に補正する。Hull-White下で解析近似でき、$a\to0$ でHo-Leeの
  $\frac12\sigma^2 t_1 t_2$ に落ちる。満期・ボラともに二乗で効く。
- 数式: $\text{CA}(t_1,t_2)=\dfrac{B(t_1,t_2)}{t_2-t_1}\!\left[B(t_1,t_2)(1-e^{-2at_1})
  +2aB(0,t_1)^2\right]\!\dfrac{\sigma^2}{4a}$, $\ B(u,v)=\frac{1-e^{-a(v-u)}}{a}$
- 関連: フォワードレート, SOFR/TONA
- 初出: S2-4

### 補間（interpolation）
- 定義: 少数のノードで与えられた値の間を関数で埋める操作。割引カーブでは
  「何を補間するか」（割引係数・その対数 $\ln DF$・ゼロレート・瞬間フォワード）の
  選択が、フォワードレートの形（段差・振動・滑らかさ）を決める。フォワードは対象量の
  微分として現れるため、価格が似ていてもフォワードは大きく違いうる。
- 数式: $f(t) = -\dfrac{d}{dt}\ln DF(t)$
- 関連: スプライン, 局所性, フォワードレート
- 初出: S2-2

### スプライン（spline）
- 定義: 区間ごとの低次多項式を、ノードで滑らかさ（値・1階・2階微分の連続）を保って
  つないだ関数。自然三次スプラインは端の2階微分を0とし、$\ln DF$ を $C^2$ で補間する
  のでフォワードは $C^1$（滑らか）になる。反面、係数が大域的に決まり非局所で、
  入力にない振動（オーバーシュート）を起こしうる。
- 数式: 区間 $[x_i,x_{i+1}]$ で3次多項式、ノードで $S,S',S''$ が連続
- 関連: 補間, 局所性, 単調性
- 初出: S2-2

### 単調性（monotonicity）
- 定義: 入力の増減の向きを補間が保つ性質。フォワード補間ではフォワードの正値・
  非振動と結びつく。三次スプラインは単調性を壊して振動しうるが、Hyman フィルタや
  単調凸法は単調性を課して振動を抑える。
- 関連: Hagan-West, スプライン
- 初出: S2-2

### 局所性（locality）
- 定義: 1点のノードの入力変化が、その近傍の区間だけに影響し、遠くの年限へ波及しない
  性質。log-DF 線形・ゼロ線形・単調凸は局所的で、自然三次スプラインは非局所（1点の
  修正がカーブ全体を揺らす）。リスク要因の年限帰属を明確にするうえで重要になる。
- 関連: 補間, Hagan-West
- 初出: S2-2

### Hagan-West（Hagan-West / monotone convex）
- 定義: フォワードを直接構成する補間法。区間フォワードを厳密に再現しつつ、正値・
  単調性・非振動・局所性を両立するよう、区間内の超過フォワードを4領域で場合分けして
  張る。log-DF 線形のフォワード段差と、三次スプラインの非局所な振動の双方を避ける
  目的で考案された。QuantLib の `MonotonicCubicNaturalSpline`（Hyman フィルタ）とは
  別系統の単調化である。
- 数式: 超過フォワード $g(x)=g(0)(1-4x+3x^2)+g(1)(-2x+3x^2)$、$\int_0^1 g\,dx=0$
- 関連: 単調性, 局所性, スプライン
- 初出: S2-2
