# 用語集 — 短期金利モデル（S5）

定義の正はこのファイル。各 notebook 末尾の用語表はここへのリンクとして扱う。
項目は S5 の各 notebook（S5-1〜S5-5）の初出順にまとまっている。

### 平均回帰（mean reversion）
- 定義: 確率変数を長期水準へ引き戻す力を持つ性質。金利モデルの基本的な性質で、
  水準から離れるほど中心へ戻すドリフトが働く。株価の発散的な挙動と対照的。
- 数式: $dr_t = a(b - r_t)\,dt + \sigma\, dW_t$（$a>0$ が回帰速度、$b$ が長期水準）
- 関連: [[アフィンモデル]]
- 初出: S5-1

### アフィンモデル（affine model）
- 定義: ゼロクーポン債価格が短期金利の指数アフィン形 $P=\exp(A(\tau)-B(\tau)r)$ で
  書ける短期金利モデル。価格づけの PDE が $A,B$ の常微分方程式2本に分離し、
  数値積分なしで解ける。Vasicek・CIR がこの族に属する。
- 数式: $P(t,T) = \exp\!\big(A(\tau) - B(\tau)\, r_t\big)$, $\tau=T-t$
- 関連: [[平均回帰]], [[ゼロクーポン債（ZCB）]]
- 初出: S5-1

### ゼロクーポン債（ZCB, zero-coupon bond）
- 定義: 途中利払いがなく満期に額面のみを払う債券。年限 $\tau$ の ZCB 価格は
  割引係数そのもので、ゼロレート $y=-\ln P/\tau$ と一対一に対応する。
- 数式: $P(t,T) = \mathbb{E}^{\mathbb{Q}}\!\big[\exp(-\int_t^T r_s\,ds)\big]$
- 関連: [[アフィンモデル]]
- 初出: S5-1

### リスクの市場価格（market price of risk）
- 定義: 実測度 $\mathbb{P}$ とリスク中立測度 $\mathbb{Q}$ をつなぐ量。Vasicek では定数 $\lambda$ と
  仮定し、実測度の長期水準 $b$ をリスク中立長期水準 $b^{\mathbb{Q}}=b+\lambda\sigma/a$ へ
  付け替える。価格づけには常に $\mathbb{Q}$ 側パラメータを使う。
- 数式: $b^{\mathbb{Q}} = b + \dfrac{\lambda \sigma}{a}$, $dW_t^{\mathbb{Q}} = dW_t^{\mathbb{P}} - \lambda\,dt$
- 関連: [[平均回帰]]
- 初出: S5-1

### 平方根過程（square-root process）
- 定義: 拡散項が水準の平方根 $\sigma\sqrt{r_t}$ に比例する確率過程。CIR モデルの
  短期金利が従う。水準が下がるほどゆらぎが小さくなり、正のドリフトが引き戻す
  ため金利が非負に保たれる。分散が水準に比例する（水準依存ボラティリティ）。
- 数式: $dr_t = a(b - r_t)\,dt + \sigma\sqrt{r_t}\;dW_t$
- 関連: [[フェラー条件]], [[非中心カイ二乗分布]], [[平均回帰]]
- 初出: S5-2

### フェラー条件（Feller condition）
- 定義: CIR 過程が原点 $r=0$ に到達しないための条件 $2ab\ge\sigma^2$。成立時は
  $r_t$ が確率1で厳密に正のまま。破れると正の確率で $0$ に到達する。定常ガンマ
  分布の shape $2ab/\sigma^2$、非中心カイ二乗の自由度 $4ab/\sigma^2\ge2$ と同値で、
  原点近傍の密度形状（$0$ で消えるか発散するか）の境目になる。
- 数式: $2ab \ge \sigma^2 \iff \dfrac{2ab}{\sigma^2}\ge 1 \iff \dfrac{4ab}{\sigma^2}\ge 2$
- 関連: [[平方根過程]], [[非中心カイ二乗分布]], [[厳密サンプリング]]
- 初出: S5-2

### 非中心カイ二乗分布（noncentral chi-squared distribution）
- 定義: CIR の遷移分布 $r_{t+\Delta}\mid r_t$ が従う分布（スケール定数倍）。自由度
  $d=4ab/\sigma^2$、非心度 $\lambda=r_t e^{-a\Delta}/c$、スケール
  $c=\sigma^2(1-e^{-a\Delta})/(4a)$。平方根過程を独立 OU 過程の2乗和として構成
  できることに由来し、$d<2$ で原点が到達可能になる。
- 数式: $r_{t+\Delta}\mid r_t = c\,\chi'^2_{d}(\lambda)$, $c=\dfrac{\sigma^2(1-e^{-a\Delta})}{4a}$
- 関連: [[平方根過程]], [[フェラー条件]], [[厳密サンプリング]]
- 初出: S5-2

### 厳密サンプリング（exact sampling）
- 定義: 遷移分布そのものから直接乱数を引き、離散化誤差ゼロで確率過程のパスを
  生成する方法。CIR では各ステップで非中心カイ二乗乱数を引く。時間刻み $dt$ に
  依存せず正しい分布を返すため、Euler 等の離散化が壊れるフェラー条件違反領域で
  特に有効。Vasicek では OU 遷移（正規分布）に相当する。
- 数式: $r_{t+\Delta}=c\,\chi'^2_{d}(\lambda)$（$d,\lambda,c$ は各ステップで再計算）
- 関連: [[非中心カイ二乗分布]], [[フェラー条件]]
- 初出: S5-2

### no-arbitrageモデル（no-arbitrage model）
- 定義: 初期の市場割引カーブ $P^M(0,\cdot)$ を厳密に再現するよう構成した金利モデル。
  Hull-White は Vasicek の定数水準を時間関数 $\theta(t)$ に置き換え、出発点で市場と
  一致させる。初期ミスプライスが混ざらないため、金利デリバティブ評価の土台になる。
  Vasicek/CIR のような均衡（endogenous）モデルと対になる概念。
- 数式: $P^{\mathrm{model}}(0,T) = P^{M}(0,T)\ \ (\forall T)$
- 関連: [[θ(t)シフト]], [[三項ツリー]], [[アフィンモデル]]
- 初出: S5-3

### 三項ツリー（trinomial tree）
- 定義: 各節点から3方向（上・中・下）へ分枝する再結合格子。Hull-White の離散化に
  使う。分枝先が3つあると平均と分散の2条件を満たしても確率に自由度が1つ残り、
  平均回帰のドリフトを格子のずらしでなく分枝確率で表せる。空間刻み
  $\Delta x=\sigma\sqrt{3\Delta t}$、上下端で分枝を切り替えて幅を打ち切る。
- 数式: $p_u=\tfrac16+\tfrac{\eta^2+\eta}{2},\ p_m=\tfrac23-\eta^2,\ p_d=\tfrac16+\tfrac{\eta^2-\eta}{2}$（$\eta$ は中心分枝先からの残差）
- 関連: [[平均回帰速度]], [[θ(t)シフト]], [[no-arbitrageモデル]]
- 初出: S5-3

### 平均回帰速度（mean reversion speed）
- 定義: 短期金利が長期水準へ引き戻される速さを表すパラメータ $a>0$。大きいほど
  回帰が強く、金利が中心から離れた状態に到達しにくい。三項ツリーでは打ち切り幅
  $j_{\max}=\lceil 0.184/(a\Delta t)\rceil$ を通じて格子の広がりを決め、$a$ が小さいほど
  格子は広く、大きいほど細く分枝の切替が早く起こる。
- 数式: $j_{\max} = \lceil 0.184 / (a\,\Delta t) \rceil$
- 関連: [[三項ツリー]], [[平均回帰]]
- 初出: S5-3

### θ(t)シフト（theta(t) shift）
- 定義: Hull-White で初期カーブに整合させるための時間依存の水準項。短期金利を
  $r(t)=x(t)+\alpha(t)$ と分解したときの決定論部分 $\alpha(t)$ を、市場フォワードに
  凸性補正を足した閉形式で与える。三項ツリーでは各時間段のシフト $\alpha_i$ を
  Arrow-Debreu 価格を使う前向き帰納で決め、$P^M(0,k\Delta t)$ を再現させる。
- 数式: $\alpha(t)=f^M(0,t)+\dfrac{\sigma^2}{2a^2}(1-e^{-at})^2$, $\ \theta(t)=\partial_t f^M(0,t)+a f^M(0,t)+\dfrac{\sigma^2}{2a}(1-e^{-2at})$
- 関連: [[no-arbitrageモデル]], [[三項ツリー]]
- 初出: S5-3

### キャリブレーション（calibration）
- 定義: モデルのパラメータを、市場で観測される価格（またはインプライドボラ）に
  最も近づくよう推定する作業。Hull-White では $a, \sigma$ をスワップション市場へ
  合わせる。目的関数（価格差かボラ差か・重み）の設計が結果を左右する。
- 数式: $\min_{a,\sigma} \sum_i w_i\,(V_i^{\mathrm{model}} - V_i^{\mathrm{mkt}})^2$
- 関連: [[Levenberg-Marquardt法]], [[識別性]], [[局所解]]
- 初出: S5-4

### Levenberg-Marquardt法（Levenberg-Marquardt）
- 定義: 非線形最小二乗問題を解くアルゴリズム。ガウス・ニュートン法と最急降下法を
  減衰パラメータで補間し、少数パラメータ・滑らかな残差で速く安定に収束する。
  スワップションキャリブレーションの定番ソルバ。
- 関連: [[キャリブレーション]]
- 初出: S5-4

### 識別性（identifiability）
- 定義: 観測データからパラメータを一意に決められる度合い。Hull-White の $a$ と
  $\sigma$ は単一オプションでは分離しにくく、満期・テナーの異なる複数セルを
  同時に使うことで識別性が上がる。面が平坦だと $a$ が定まりにくい。
- 関連: [[キャリブレーション]], [[局所解]]
- 初出: S5-4

### 局所解（local minimum）
- 定義: 目的関数の谷のうち、大域最小ではないもの。非線形最小化は初期値によって
  局所解に落ちうる。初期値を変えた再現性確認・パラメータ範囲の付与・面の情報量
  確保が対策になる。
- 関連: [[キャリブレーション]], [[識別性]]
- 初出: S5-4

### ATMボラ（at-the-money volatility）
- 定義: 行使レートをフォワードスワップレートに一致させたときのインプライド
  ボラティリティ。本源的価値がゼロで時間価値だけになるため、ボラ情報が最も
  素直に表れる。スワップション市場はこの ATM ボラを満期×テナーの面でクォート
  する。
- 関連: [[キャリブレーション]]
- 初出: S5-4

### モデルリスク（model risk）
- 定義: モデルの誤りや誤用が、誤った価格・リスク判断・意思決定を通じて損失を
  生む可能性。同一商品でもモデル選択で価格・Greeks が食い違うことが典型で、
  カーブ整合の有無・分布特性・自由度の差がその源泉になる。
- 関連: [[モデルバリデーション]], [[ベンチマークモデル]], [[感応度分析]]
- 初出: S5-5

### モデルバリデーション（model validation）
- 定義: モデルの前提・実装・出力を開発者から独立に検証し、妥当性と限界を
  評価する営み。FRB/OCC の SR 11-7 が事実上の標準で、概念的健全性の評価・
  継続的モニタリング・アウトカム分析（ベンチマーク比較・バックテスト）の
  3本柱で行う。
- 関連: [[モデルリスク]], [[ベンチマークモデル]], [[感応度分析]]
- 初出: S5-5

### ベンチマークモデル（benchmark model）
- 定義: 検証対象と突き合わせるための独立な基準モデル・別実装。唯一の正解が
  無い評価で、差を説明できる状態を保つために用いる（本シリーズでは QuantLib の
  対応クラスや他モデルとの相互比較）。
- 関連: [[モデルバリデーション]], [[モデルリスク]]
- 初出: S5-5

### 感応度分析（sensitivity analysis）
- 定義: 入力（カーブ・ボラティリティ・平均回帰速度など）を振って出力の変化を
  測り、その変化が経済的に説明できるかを確認する手法。変化が大きい入力ほど
  モデルリスクの所在となる。Greeks の算出（bump-and-revalue）もこれに含む。
- 数式: $\partial V/\partial x \approx (V(x+\Delta)-V(x-\Delta))/(2\Delta)$（共通乱数で中心差分）
- 関連: [[モデルバリデーション]], [[モデルリスク]]
- 初出: S5-5

### G2++／LMM（G2++ / LIBOR Market Model）
- 定義: 1ファクター短期金利モデルの表現力不足を補う多因子・カーブ全体モデル。
  G2++ は相関する2つの Gaussian ファクターでカーブの水準とスロープを独立に
  動かす（Hull-White の自然な拡張で、解析トラクタビリティを保持）。LMM（BGM）は
  観測可能な離散フォワード金利群を直接モデル化し、キャップ・スワップションの
  市場整合キャリブレーションと経路依存エキゾチックに用いる。HJM は瞬間フォワード
  カーブ全体をモデル化する母体的枠組みで、無裁定がドリフト条件として現れる。
- 関連: [[モデルリスク]], [[キャリブレーション]]
- 初出: S5-5
