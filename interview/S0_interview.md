# S0 面接想定問答

環境・データ基盤の回。口頭で説明できる状態を目標にする。模範解答は要点のみ。

1. **numpy/scipy と QuantLib をどう使い分けるか。**
   モデルは numpy/scipy で自作し、QuantLib は検証用ベンチマークに使う。面接で
   問われるのは導出・実装力なので、ライブラリ呼び出しに頼らない。

2. **ライブラリのレイヤーを分ける利点は。**
   依存の向きを一方向にすると変更の影響範囲が読める。層ごとにテストを持てる。

3. **「10年金利」と言われたとき、何を確認するか。**
   ソースと種類。FRED の CMT はパー利回りのグリッド、JSDA は銘柄別利回り。
   カーブの入力にするパー利回りと、割引に使うゼロレートは別物。

4. **パー利回りとゼロレートの違いは。**
   パー利回りはその年限の標準クーポン債が額面で取引される利回り。ゼロレートは
   単一の将来キャッシュフローの割引に使うレート。パー→ゼロは剥ぎ取り（S2）。

5. **財務省の国債金利情報で銘柄別の相対価値分析はできるか。**
   できない。財務省は年限グリッドのパー利回りのみ。銘柄別は JSDA の売買参考
   統計値を使う。

6. **API キーをどう扱うか。**
   `.env` に置き `.gitignore` で除外。コードや出力に書かない。公開リポジトリでは
   push 前の検知（pre-commit）で漏えいを止める。

7. **時系列データの欠損をどう処理するか。**
   用途による。分析用途では観測のある日だけ残す。前値埋めは休場を観測のように
   見せ、日次変化のボラを下げるため裾を歪める。

8. **再現性をどう担保するか。**
   取得結果をキャッシュし、乱数は seed 固定。鍵の無い環境ではキャッシュか合成
   サンプルで同じ結果を返す。
