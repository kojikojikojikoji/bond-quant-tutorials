# S1 面接想定問答

債券の基礎の回。口頭で導出・説明できる状態を目標にする。

1. **半年複利5%と等価な連続複利レートは。**
   $r_c = 2\ln(1+0.05/2) = 4.94\%$。割引係数を保つ変換で、$t$ に依存しない。

2. **連続複利をモデルで使う理由は。**
   割引が $e^{-rt}$ と積分・微分で扱え、確率過程の枠組みに乗せやすい。

3. **ACT/360 と ACT/365F の使い分けは。**
   ACT/360 は USD/EUR マネーマーケット、ACT/365F は JGB や GBP マネーマーケット。
   分母が違うので同じ実日数でも年数が変わる。

4. **30/360 が各期のクーポンを同額にできる理由は。**
   月を30日・年を360日に固定し、各半年を必ず180/360＝0.5にするため。

5. **YTM の前提は。**
   満期保有、全クーポンを同じ YTM で再投資、無デフォルト。現実には再投資
   利回りが変わるため実現リターンと乖離する。

6. **単利利回りと複利 YTM はなぜ一致しないか。符号は。**
   単利は再投資と複利を無視する。ディスカウント債では単利が高め、プレミアム債
   では低めに出る。

7. **価格-利回り曲線の凸性とは。何に効くか。**
   曲線が下に凸で、デュレーションの一次近似は真の価格を下側に外す。利回り低下の
   値上がりが上昇の値下がりを上回る非対称を生む。

8. **クリーン価格とダーティ価格の違いは。どちらで受け渡すか。**
   ダーティ＝クリーン＋経過利子。受け渡すのはダーティ。表示はクリーンで、受渡日
   による鋸歯状の変動を除いて比較しやすくする。

9. **債券の経過利子で ACT/ACT の ISDA と ISMA はどちらを使うか。違いは。**
   債券は ISMA/Bond（実日数の期間比）。ISDA は暦年ごとに365/366で按分するため、
   うるう年境界をまたぐ期間で期間比に使うとずれる（最大 5e-4 程度）。

10. **T+1 と T+2 の市場は。経過利子はどの日を基準にするか。**
    UST・Gilt が T+1、JGB・Bund が T+2。経過利子は約定日ではなく受渡日基準。
