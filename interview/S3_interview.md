# S3 面接想定問答

リスク指標の回。口頭で導出・説明できる状態を目標にする。

1. **マコーレー・修正・実効デュレーションの違いは。**
   マコーレーは加重平均回収期間、修正は $D_{mac}/(1+y/f)$ で価格感応度、実効は
   バンプ再評価による数値微分でCFが利回り依存でも定義できる。

2. **コンベクシティは何を表すか。なぜ効くか。**
   価格-利回り曲線の曲率。デュレーションの一次近似は真の価格を下に外し、その
   差の主要項がコンベクシティ。大きく金利が動くほど効く。

3. **DV01 とは。デュレーションとの関係は。**
   1bp あたりの価格変化額。$DV01 = D_{mod}\cdot P\cdot 10^{-4}$。金額ベースなので
   異なる債券のリスクを足し合わせられる。

4. **DV01ヘッジの限界は。**
   平行シフトしか消せない。カーブがスティープ/フラット/ツイストすると残存PnLが
   出る。テナー別のKRDで見る必要がある。

5. **キーレートデュレーションとは。合計は何になるか。**
   テナー別にカーブをバンプした感応度。合計は全体の修正デュレーション（並行DV01）
   に一致する。バーベルとブレットはデュレーションが同じでもKRDプロファイルが違う。

6. **カーブ変動のPCAで何が分かるか。**
   第1主成分がレベル（平行）、第2がスロープ、第3が曲率。3因子で分散の95〜99%を
   説明できる。リスクを少数の直交因子に圧縮できる。

7. **VaR と Expected Shortfall の違いは。**
   VaR は損失の分位点、ES はその超過部分（最悪 1−α 質量）の平均。ES は裾の大きさ
   まで見る。

8. **なぜ ES が VaR より好まれるか（劣加法性）。**
   VaR は分散投資でリスクが増えるように見える反例があり劣加法性を満たさない。ES は
   コヒーレントで劣加法性を満たす。バーゼルも 97.5% ES へ移行した。

9. **VaR の3つの計算方式の違いは。**
   ヒストリカル（過去分布そのまま）、パラメトリック（正規等を仮定）、モンテカルロ
   （モデルからパスを生成）。裾が厚いとヒストリカルとパラメトリックが乖離する。

10. **VaR モデルの妥当性はどう検証するか。**
    Kupiec の POF 検定。実際の例外数が想定被覆率 (1−α) と整合するかを尤度比で調べる。
