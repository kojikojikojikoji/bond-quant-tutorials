# S4 確率過程の数学

## シリーズ概要

デリバティブ評価・金利モデル・リスク計測の土台になる確率過程を、数式と数値実験の両輪で身につけるシリーズです。ブラウン運動から始めて、伊藤の公式、SDE の数値解法、モンテカルロ法、そしてリスク中立評価までを順に積み上げます。各テーマは「理論を省かずに書く」一方で、必ず解析解や QuantLib と数値で突き合わせ、正しく実装できているかを目で確かめられる構成にしています。後段の S5（短期金利モデル）・S6（金利デリバティブ）・S8（証券化の MC-OAS）は、ここで作る道具立てをそのまま前提にします。

## 学ぶこと

- ランダムウォークのスケーリング極限としてのブラウン運動と、その性質（独立増分・連続だが微分不可能・非有界変動）。二次変分 $\sum(\Delta W)^2 \to t$、すなわち $(dW)^2=dt$ の意味。
- 確率積分（伊藤積分）の左端点評価とマルチンゲール性、伊藤の公式の使い方。幾何ブラウン運動（GBM）とオルンシュタイン・ウーレンベック（OU）過程の厳密解。
- SDE の離散化（Euler-Maruyama / Milstein）と、強収束・弱収束の違い。CIR 過程の負値問題と full truncation による修正。
- モンテカルロ推定量の標準誤差と $1/\sqrt{n}$ 収束、分散削減（対照変量・制御変量・準乱数）、グリークス計算のバンプ法と pathwise 法。
- リスク中立評価の根拠（複製と無裁定）、リスク中立測度と実測度の違い、Girsanov の定理、ニュメレール変換とリスクの市場価格。

## 実務での使われ方

ここで扱う数学は、そのまま値付けとリスク計測のエンジンになります。銀行の金利デスククオンツはブラウン運動のパス生成器の上に短期金利モデルを組み、伊藤の公式で価格式やヘッジ比率を導き、モンテカルロや SDE ソルバーでエキゾチックを評価します。リスク中立評価とニュメレール変換は、キャップ・スワップションの値付け（S6）に不可欠な機械です。ヘッジファンドの相対価値戦略は OU 過程の平均回帰を骨格にし、リスク管理は同じシミュレーション基盤で VaR/ES を計算します。分散削減は「同じ精度を、より少ない計算資源で」達成するための道具で、計算コストと応答速度に直結します。各 notebook 冒頭の「実務での位置づけ（ファンドはどう稼ぐか）」で、この対応を具体的に説明しています。全体像は `docs/債券ファンドの業務.md` を参照してください。

## 前提

## 使う bondlab モジュール

`bondlab.sim` を主に使います。

| 関数 | 役割 |
|---|---|
| `brownian_paths` | ブラウン運動のパスをベクトル化して生成（seed 固定） |
| `simulate_sde` | ドリフト・拡散係数を渡して SDE を数値的に解く（Euler-Maruyama） |
| `mc_stats` | モンテカルロ標本の平均・標準誤差・信頼区間を計算 |
| `control_variate` | 制御変量法による分散削減（最適係数の推定を含む） |

## 検証方針

- 解析解（GBM・OU・Black-Scholes）と自作コードの数値解を突き合わせ、実装の正しさを確認します。
- QuantLib（OU 過程・欧州オプション）を独立の基準として突合します。
- 収束次数を数値で確かめます。強収束は Euler-Maruyama で約 0.5、Milstein で約 1.0、モンテカルロ誤差は $1/\sqrt{n}$。log-log 回帰で傾きを実測します。
- 乱数はすべて seed 固定とし、結果が再現できるようにします。
