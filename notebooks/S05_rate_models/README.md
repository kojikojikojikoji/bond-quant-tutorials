# S5 短期金利モデル

## このシリーズの狙い

短期金利（instantaneous short rate）$r_t$ の確率過程を1本の SDE で表し、そこから
割引債・イールドカーブ・金利オプションの価格までを一貫して導く「短期金利モデル」を
扱います。銀行の金利デスクでは、コーラブル債・スワップション・キャンセラブル
スワップといった金利の将来経路に依存する商品を値付けし、ヘッジするために、この種の
モデルを日々キャリブレーションして使います。S5 は Vasicek → CIR → Hull-White と
段階を追い、最後にモデルの選択そのものがリスク源になること（モデルリスク）まで
つなげます。前半の山場は S5-4 のスワップションキャリブレーションです。

## 学ぶこと

- 平均回帰型の SDE を、回帰速度・長期水準・ボラティリティの3パラメータで読み解く
- アフィン期間構造から、ゼロクーポン債（ZCB）価格の解析解をゼロベースで導出する
- 短期金利の分布（Vasicek＝正規／CIR＝非心カイ二乗）と、負金利・非負性の帰結
- 厳密サンプリングと離散化（オイラー）の違い、フェラー条件が壊す原点近傍の挙動
- 時間依存ドリフト $\theta(t)$ で初期カーブを厳密再現する無裁定モデルの仕組み
- Hull-White の三項ツリーを2段階法でスクラッチ実装し、初期カーブを再現する
- スワップション市場（満期×テナー、ATM、Black ボラ）へのキャリブレーション
- Vasicek／CIR／Hull-White の比較軸と、モデルバリデーションの実務枠組み

## 実務での使われ方

金利デスクのクオンツ（フロント）は、カーブ構築（S2）で得た割引係数を入力に、短期金利
モデルを組んで経路依存商品を評価し、デルタ・ベガ・バケット感応度を出します。デスクは
その値付けでマーケットメイクの両面クォートを出してスプレッドを稼ぎ、ヘッジで金利水準・
ボラのリスクを落とします。モデルの $a,\sigma$ はスワップション市場へのキャリブレーション
で毎営業日更新されます。これらをリスク管理クオンツが DV01・KRD・VaR で監視し、
モデルバリデーション（独立検証）が承認する、という分業になっています。

負金利を許すか否か、初期カーブを厳密に合わせるか、金利の非負性を課すかという各モデルの
仮定は、低金利・マイナス金利局面での価格を大きく分けます。どのモデルを選ぶかが損益と
リスクに直結するため、S5-5 ではモデル選択そのものをリスクとして数値化します。

## 前提

- S2（イールドカーブ構築：割引係数・フォワードレート）
- S4（確率解析の基礎：SDE・伊藤の補題・リスク中立測度・リスクの市場価格）
- Python（NumPy）、および検証用に QuantLib

## 使う bondlab モジュール

`bondlab.models` の以下のクラスを使います。

- `Vasicek` — Vasicek モデル（ZCB 解析解・イールドカーブ・サンプリング）
- `CIR` — CIR モデル（ZCB 解析解・厳密サンプリング）
- `HullWhite` — Hull-White モデル（初期カーブ再現・解析 ZCB・ツリー評価）
- `DiscountCurve` — 初期割引カーブ（$\theta(t)$ フィットやカーブ入力に使用）

## 検証方針

- Vasicek・CIR の ZCB 解析解を QuantLib と機械精度で突合する
- CIR は厳密サンプリング（非心カイ二乗）と離散化を比較し、フェラー条件を破る場合の
  離散化の破綻を定量化する
- Hull-White は初期カーブを全テナー <1bp で再現し、三項ツリー上の債券評価が解析 ZCB・
  QuantLib の解析値へ収束することを確認する
- S5-4 のキャリブレーションは、合成ボラ面から既知パラメータの復元を主検証に据え、
  初期値に対する安定性を評価する
- S5-5 は3モデルの相互比較・QuantLib ベンチマーク・カーブ再現誤差を、FRB SR 11-7 の
  概念的健全性・継続的モニタリング・アウトカム分析の3本柱に沿って行う
