# S7 クレジット

社債・CDS・カウンターパーティ信用リスクといった「個社の信用力（ミクロ）」を扱う
シリーズです。金利（レート）を主軸にした S1〜S6 に対し、S7 は発行体がデフォルトする
確率と、それが価格・スプレッド・デリバティブ評価にどう反映されるかを、スクラッチ実装と
`bondlab.credit` の突合を通じて身につけます。

## このシリーズで学ぶこと

- デフォルトを強度 $\lambda$ のポアソン過程として表す誘導型（ハザード）モデルと、
  生存確率 $S(t)=\exp(-\int_0^t\lambda)$ によるリスキー割引
- CDS のプライシング（プレミアム／プロテクションの両レッグ）、CDS タームストラクチャ
  からのハザードカーブ・ブートストラップ、クレジット DV01（CS01）
- 企業のバランスシートからデフォルトを内生的に説明する Merton 構造モデル
  （距離to default・PD・KMV 流の実務化）
- キャッシュ債のスプレッド4指標（G-spread / I-spread / ASW / OAS）と Z スプレッド、
  IG／HY 時系列からの景気局面判定
- 金利モンテカルロ（S5）とハザードカーブ（S7）を結合した CVA（カウンターパーティ
  信用評価調整）とエクスポージャープロファイル

## 実務での使われ方

クレジットデスク（セルサイド）は社債・CDS を両面クォートで値付けし、在庫の信用リスクを
ハザードカーブ上で評価・ヘッジします。クレジットヘッジファンドは、キャリー（プロテクション
売り・社債保有）、方向性、相対価値（CDS-ボンド・ベーシス、クレジットカーブ取引、
キャピタルストラクチャー・アービトラージ）で稼ぎます。運用会社は格付・セクター・年限で
切ったスプレッドの相対価値と、KMV 流の EDF による信用スクリーニングでアルファを狙います。
XVA デスクは、CDS からブートストラップした生存確率と将来エクスポージャーを掛け合わせて
CVA を価格化し、金利デルタ・ベガ、信用デルタをヘッジします。各 notebook 冒頭の
「実務での位置づけ（ファンドはどう稼ぐか）」で、`docs/債券ファンドの業務.md` の収益源
（マーケットメイク・相対価値・キャリー・マクロ・XVA）に結びつけています。

## 前提

- S2（イールドカーブ／割引）、S5（金利モンテカルロ、S7-5 で必須）を先に読んでいると
  理解が早いです。S7-1〜S7-3 は強度・構造モデルの流れ、S7-4 はキャッシュ債スプレッドで
  独立に読めます。
- Python（NumPy / SciPy / matplotlib）。S7-2 の QuantLib 突合には `QuantLib-Python` を使います。
- 実行は `bondlab` をインポートできる状態で（リポジトリルートで `PYTHONPATH=$PWD`）。

## 使う bondlab モジュール

- `bondlab.credit`
  - `HazardCurve` … 区分定数ハザードから生存確率・リスキー割引を返す（S7-1, S7-5）
  - `cds_par_spread` … CDS のフェアスプレッド（経過保険料込み）（S7-2）
  - `bootstrap_hazard` … CDS タームストラクチャからハザードカーブを剥ぎ取る（S7-2, S7-5）
  - `Merton`（距離to default・PD・資産価値/ボラの逆算）（S7-3）
- S7-4 は `bondlab` の債券・ゼロカーブ（G/I/Z スプレッド計算機を自作）を利用
- S7-5 は S5 の Hull-White 金利モンテカルロと `bondlab.credit` を結合

## 検証方針

各 notebook は、スクラッチ実装を `bondlab` および外部の基準と突き合わせて正しさを確認します。

- S7-1：$\lambda$ 一定の解析解 $S(t)=e^{-\lambda t}$ と自作カーブの一致、近似式
  $s\approx\lambda(1-R)$ の精度・限界をパラメータ範囲で検証
- S7-2：自作 CDS エンジンを QuantLib の `CreditDefaultSwap`＋`MidPointCdsEngine` と
  約 0.02bp 以内で突合、ハザード・ブートストラップは往復（カーブ→par spread→再カーブ）で
  厳密再現、CS01 は数値微分で確認
- S7-3：Black–Scholes 一致・恒等式（$B_0=V_0-E_0$）、株価/株式ボラからの反復逆算の収束確認
- S7-4：スプレッド指標間の整合性（G と I の差＝スワップスプレッド、Z と G の大小）を内部検証
- S7-5：独立性の仮定のもとで CVA を離散和に落とし、エクスポージャープロファイルの形状
  （中間の山）と、生存確率・割引エクスポージャーの分解を確認
