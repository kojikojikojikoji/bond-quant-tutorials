# S8 証券化商品（オプション回・米系運用志望向け）

住宅ローンをまとめて証券にした MBS（mortgage-backed security）を題材に、
プリペイメント（期限前償還）とネガティブコンベクシティ、そして
モンテカルロ OAS までを一通り自作するシリーズです。国債やスワップの延長では
説明しきれない「借り手が握るコールオプション」を、キャッシュフローの組み立てから
評価・ヘッジまで数値で追います。

## このシリーズで学ぶこと

- パススルー証券のキャッシュフローを利息・予定元本・期限前償還の3成分に分解して構成する
- CPR・SMM・PSA の関係と、金利差に反応するプリペイメントの S字カーブ・バーンアウト
- 金利 $\pm$100bp での価格変化から読むネガティブコンベクシティ（負のコンベクシティ）
- パス依存商品を評価するモンテカルロ OAS と、OAS・Zスプレッド・オプションコストの分解
- 日本の機構MBS（JHF）の制度（超過担保・団信・月次公表）と、米国より低いプリペイメント感応度

## 実務での使われ方

米系運用会社の MBS デスクやコア債券ファンドは、MBS を「エージェンシー保証で信用リスクは薄いが、
プリペイメント次第で損益が動く商品」として扱います。プールごとに将来 CPR を読み分けて仕入れ値を変える
銘柄選別（プールペイアップ）、OAS の拡大・縮小を狙ったスプレッドの相対価値、そして金利が下がる局面で
償還が加速して高クーポンを失うネガティブコンベクシティのヘッジ（スワップション・国債先物）が主な稼ぎ・
損失回避のレバーです。MBS を保有することは実質的に金利ボラティリティを売っているのに近く、
デスクは自前の OAS が示すインプライドボラティリティをスワップション市場と突き合わせて値付けします。
日本の機構MBS は団信による金利に鈍感な非自発的償還が期限前償還の床を作るため、米国型のボラティリティ取引
よりキャリーと ALM 適合に寄った持ち方になります。全体像は `docs/債券ファンドの業務.md`（運用会社・ALM、
相対価値、キャリー）を土台にしています。

## 使う bondlab モジュール

`bondlab.mbs` のプリペイメント・キャッシュフロー系を使います。

- `psa_cpr` — PSA モデルのローンエイジ別 CPR（age30 で 6% 到達の ramp）
- `cpr_to_smm` / `smm_to_cpr` — 年率 CPR と月率 SMM の相互変換
- `mbs_cashflows` — 利息・予定元本・期限前償還に分けたパススルーのキャッシュフロー生成
- `weighted_average_life` — 元本返済のタイミングを加重平均した WAL

## 検証方針

- PSA ramp と WAL は自作の独立計算で検算し、`bondlab.mbs` の結果と一致することを確認する
- MC-OAS はプリペイメントをゼロにすると通常の Zスプレッドに一致することを確認し、評価エンジンの妥当性検証とする
- S8-3 では米国の PSA/計量モデルを日本の実績（を模した合成データ）に当てはめ、フィットの悪さを定量化する

S8 はチュートリアルのコアではなくオプション回です。証券化・米系運用を志望する場合に選択してください。
